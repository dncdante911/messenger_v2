# Windows Messenger — Feature Roadmap

Перенос функционала с Android-версии + UI-улучшения.
Каждая задача — конкретный, тестируемый кусок работы.

---

## Batch 1 — Визуал + поиск + сохранёнки ✅

| # | Задача | Статус |
|---|--------|--------|
| 1 | **Тема Ocean Night** — мягкая тёмная navy палитра, точечный паттерн как в TG, sky-blue акцент | ✅ |
| 2 | **Глобальный поиск** — люди, группы, каналы, debounce 350 ms | ✅ |
| 3 | **Сохранённые сообщения** — кнопка 🔖 в меню бабла, отдельный экран, API sync | ✅ |

---

## Batch 2 — Качество жизни в чатах ✅

| # | Задача | Статус |
|---|--------|--------|
| 4 | **Заметки** — перенос с Android: composer внизу, карточки заметок, хранилище, API sync | ✅ |
| 5 | **Настройки как оверлей** — полноэкранный overlay как в Telegram Desktop, кнопка ✕ | ✅ |
| 6 | **История звонков** — полноценный экран с аватарами, типом (входящий/исходящий/пропущенный), callback кнопкой | ✅ |

## Batch 3 — Профиль, статус и Stories ✅

| # | Задача | Статус |
|---|--------|--------|
| 7 | **Расширенный профиль** — поля: дата рождения, город, пол, работа, учёба, телефон, сайт, bio | ✅ |
| 8 | **Социальные ссылки** — Facebook, Twitter, Instagram, LinkedIn, YouTube в отдельной вкладке настроек | ✅ |

| 10 | **Настройки уведомлений** — 10 переключателей (email, лайки, комментарии, подписчики…), server sync | ✅ |
| 11 | **Редизайн Stories** — круглые аватарки в стиле Instagram, карточки, просмотрщик с реакциями 6 эмодзи, комментарии с панелью, удаление своих историй | ✅ |
