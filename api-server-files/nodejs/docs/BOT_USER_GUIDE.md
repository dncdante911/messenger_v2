# Боты в WorldMates — Руководство пользователя

---

## Что такое боты?

Боты — это автоматические собеседники в WorldMates. Они могут:
- Отвечать на вопросы
- Помогать с задачами
- Проводить опросы
- Отправлять новости и уведомления

Боты выглядят как обычные пользователи, но их имя заканчивается на **_bot**.

---

## Как найти бота?

1. Откройте поиск в мессенджере
2. Введите имя бота (например `wallybot`)
3. Нажмите на бота в результатах поиска
4. Откроется чат

---

## WallyBot — твой помощник

**@wallybot** — официальный бот WorldMates. Он помогает:
- Создавать собственных ботов
- Управлять ботами через чат
- Отвечать на вопросы

### Команды WallyBot:

| Команда | Что делает |
|---------|-----------|
| `/start` | Главное меню |
| `/help` | Список команд |
| `/newbot` | Создать нового бота |
| `/mybots` | Твои боты |
| `/editbot` | Изменить настройки бота |
| `/deletebot` | Удалить бота |
| `/token` | Получить токен бота |
| `/learn` | Научить WallyBot чему-то новому |
| `/ask` | Задать вопрос WallyBot |

---

## Как создать своего бота?

### Шаг 1
Открой чат с **@wallybot** и напиши `/newbot`

### Шаг 2
Введи **отображаемое имя** (например: "Мой Бот")

### Шаг 3
Введи **username** — должен заканчиваться на `_bot`
*(например: `my_helper_bot`)*

### Шаг 4
Введи **описание** или пропусти (`/skip`)

### Готово!
WallyBot покажет тебе **Bot Token** — сохрани его! Этот токен нужен для программирования бота.

---

## Как общаться с ботом?

1. Открой чат с ботом
2. Напиши сообщение или нажми кнопку
3. Бот ответит автоматически

**Команды** начинаются с `/` (например `/start`, `/help`)

**Кнопки** появляются под сообщениями — нажми на нужную

---

## Что умеет WallyBot?

WallyBot можно **обучить** отвечать на любые вопросы:

1. Напиши `/learn`
2. Введи ключевое слово (например: "контакты")
3. Введи ответ (например: "Email: support@worldmates.club")

Теперь когда кто-то напишет "контакты" — WallyBot ответит автоматически!

Чтобы удалить ответ: напиши `/forget` и выбери нужный.

---

## Безопасность

- Никому не передавай **Bot Token** — он даёт полный контроль над ботом
- Если токен скомпрометирован — обнови его через `/token` → "Обновить токен"
- Боты не имеют доступа к твоим личным данным без разрешения

---

## Поддержка

Если бот не работает или ведёт себя неправильно:
- Напиши `/start` — перезапустит диалог
- Обратись к создателю бота или в поддержку WorldMates
