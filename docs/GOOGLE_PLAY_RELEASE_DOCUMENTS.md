4. ШИФРОВАНИЕ И БЕЗОПАСНОСТЬ ДАННЫХ

4.1 Шифрование сообщений
WorldMates Messenger использует гибридную систему шифрования:
• AES-256-GCM: шифрование сквозного типа для Android-устройств
• Уникальный вектор инициализации (IV) для каждого сообщения
• Тег аутентификации (128-bit) для проверки целостности данных

4.2 Шифрование при передаче
• Все соединения с сервером защищены TLS 1.3
• WebSocket-соединения используют WSS (WebSocket Secure)
• Видеозвонки используют DTLS/SRTP через WebRTC

4.3 Хранение данных
• Пароли хранятся в виде хешей (bcrypt)
• Медиафайлы хранятся на наших серверах в зашифрованном виде
• Токены сессий имеют ограниченный срок действия

4.4 Ограничения шифрования
Настоящая версия приложения использует шифрование при передаче. Полное
сквозное шифрование (E2EE) находится в стадии активации.

## 10. ДЕКЛАРАЦИЯ РАЗРЕШЕНИЙ ANDROID

Google Play требует объяснения использования каждого чувствительного разрешения.

### Обязательные разрешения и их обоснование

| Разрешение | Назначение | Обоснование для пользователя |
|-----------|-----------|------------------------------|
| `INTERNET` | Сетевые запросы | Необходимо для отправки/получения сообщений |
| `ACCESS_NETWORK_STATE` | Проверка соединения | Адаптация к типу сети (WiFi/4G) |
| `CAMERA` | Видеозвонки, фото | Необходимо для видеозвонков и отправки фото |
| `RECORD_AUDIO` | Голосовые сообщения, звонки | Запись голосовых сообщений и аудиозвонки |
| `READ_MEDIA_IMAGES` | Отправка фото | Выбор фотографий из галереи |
| `READ_MEDIA_VIDEO` | Отправка видео | Выбор видео из галереи |
| `READ_MEDIA_AUDIO` | Отправка аудио | Выбор аудиофайлов для отправки |
| `MODIFY_AUDIO_SETTINGS` | Управление звуком | Регулировка громкости во время звонков |
| `ACCESS_FINE_LOCATION` | Geo Discovery | Поиск людей рядом (по запросу) |
| `ACCESS_COARSE_LOCATION` | Приблизительная геолокация | Определение города пользователя |
| `RECEIVE_BOOT_COMPLETED` | Автозапуск уведомлений | Получение уведомлений после перезагрузки |
| `POST_NOTIFICATIONS` | Push-уведомления | Отображение уведомлений о новых сообщениях |
| `VIBRATE` | Вибрация | Вибрация при входящих сообщениях |
| `READ_CONTACTS` | Синхронизация контактов | Поиск друзей среди ваших контактов |
| `FOREGROUND_SERVICE` | Фоновые уведомления | Обнаружение входящих звонков в фоне |
| `USE_BIOMETRIC` | Биометрия | Разблокировка приложения по отпечатку |
| `USE_FINGERPRINT` | Отпечаток пальца | Биометрическая аутентификация |

### Разрешения, требующие специального одобрения Google

| Разрешение | Тип | Действие |
|-----------|-----|---------|
| `READ_CONTACTS` | Чувствительное | Объяснить: используется только для поиска друзей |
| `ACCESS_FINE_LOCATION` | Чувствительное | Объяснить: только для функции Geo Discovery |
| `RECORD_AUDIO` | Чувствительное | Объяснить: только для звонков и голосовых сообщений |
| `CAMERA` | Чувствительное | Объяснить: только для видеозвонков и фото |
