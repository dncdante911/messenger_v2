## Юрисдикция и соответствие

Документы разработаны с учётом:
- **Законодательство Украины** (основная юрисдикция, г. Днепр)
- **GDPR** (Регламент ЕС 2016/679) — для пользователей ЕС
- **Директива NIS2** (EU) 2022/2555
- **ePrivacy Directive** 2002/58/EC

## Важные замечания

1. **Представитель в ЕС (ст. 27 GDPR)** — обязателен до начала работы с пользователями ЕС. Рекомендуется привлечь специализированную компанию-представителя в одном из государств-членов ЕС.

2. **DPO** — назначение уполномоченного по защите данных может быть обязательным в зависимости от масштаба обработки данных (ст. 37 GDPR).

3. **Перевод на украинский язык** — официальные документы, публикуемые на территории Украины, должны быть доступны на украинском языке в соответствии с Законом Украины «Об обеспечении функционирования украинского языка как государственного».

4. **Перевод на языки ЕС** — рекомендуется перевод как минимум на английский язык, а для целевых рынков — на языки соответствующих государств-членов ЕС.

5. **Юридическая проверка** — перед публикацией рекомендуется провести проверку документов квалифицированным юристом, специализирующимся на праве в сфере IT и защиты данных.
